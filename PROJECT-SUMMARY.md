```bash
git clone https://github.com/liangjie559567/axiom-omc-integration.git
cd axiom-omc-integration
npm install
npm test
```

### 作为 Claude Code 插件
```bash
git clone https://github.com/liangjie559567/axiom-omc-integration.git \
  ~/.claude/plugins/axiom-omc
# 在 Claude Code 中: /plugin activate axiom-omc
```

## 🎯 核心特性

### 1. Agent 系统
32 个专业 Agent，分布在 6 个 Lane：
- **Architect Lane** (4) - 架构设计
- **Executor Lane** (5) - 代码实现
- **Reviewer Lane** (3) - 代码审查
- **Optimizer Lane** (3) - 性能优化
- **Documenter Lane** (3) - 文档编写
- **Tester Lane** (4) - 测试

### 2. 命令系统
25 个 CLI 命令，5 个类别：
- **Agent 命令** (6) - Agent 管理
- **Workflow 命令** (7) - 工作流管理
- **Memory 命令** (5) - 记忆管理
- **Sync 命令** (4) - 状态同步
- **Plugin 命令** (3) - 插件管理

### 3. 记忆系统
- **决策管理** - 追踪和查询决策
- **知识图谱** - 构建知识关系
- **模式提取** - 自动学习模式
- **持久化** - JSON 文件存储

### 4. 工作流
- **Axiom 工作流** - 3 阶段（Draft, Review, Implement）
- **OMC 工作流** - 5 阶段（Planning, Design, Implementation, Testing, Deployment）
- **自定义工作流** - 灵活配置
