# /start - 启动 Axiom 系统并加载上下文

## 描述

启动 Axiom 智能决策系统，初始化记忆管理器、知识图谱和命令路由器，加载用户偏好和活动上下文。

## 使用方式

```
/start
```

## 功能

此技能会执行以下操作：

1. **初始化记忆管理器** - 加载历史决策记录和用户偏好
2. **初始化知识图谱** - 加载项目知识和模式库
3. **初始化命令路由器** - 配置 Axiom 和 OMC 命令路由规则
4. **加载用户偏好** - 恢复用户的个性化设置
5. **加载活动上下文** - 恢复当前工作上下文
6. **生成启动报告** - 显示系统状态和统计信息

## 输出

启动报告包含：

- **组件状态** - 各核心组件的初始化状态
- **记忆统计** - 决策记录数、用户偏好数、活动上下文数
- **知识图谱统计** - 节点数、边数、决策节点数、模式节点数
- **用户偏好** - 最近的用户偏好设置
- **活动上下文** - 当前活动的工作上下文
- **最近决策** - 最近 10 个决策记录

## 注意事项

- 首次使用时，系统会自动创建必要的目录和文件
- 如果某个组件初始化失败，系统会显示警告但继续启动其他组件
- 启动报告会自动保存到 `.agent/logs/startup_report.json` 和 `.agent/logs/startup_report.md`
