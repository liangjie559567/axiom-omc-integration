# Axiom-OMC Integration 插件安装指南

## 快速安装

### 从 GitHub 安装（推荐）

```bash
# 1. 添加插件市场
/plugin marketplace add liangjie559567/axiom-omc-integration

# 2. 安装插件
/plugin install axiom-omc@axiom-omc-integration
```

### 验证安装

```bash
# 查看已安装插件
/plugin

# 测试插件命令
/axiom-omc:start
```

## 可用功能

- 32个专业代理
- 工作流编排
- 状态同步
- 记忆管理
- CLI系统

## 更新插件

```bash
/plugin marketplace update axiom-omc-integration
```

## 卸载

```bash
/plugin uninstall axiom-omc@axiom-omc-integration
```
