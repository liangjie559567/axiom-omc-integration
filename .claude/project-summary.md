## 🏗️ 系统架构

### 核心模块

```
┌─────────────────────────────────────────────────┐
│           CLI Commands (命令接口)               │
├─────────────────────────────────────────────────┤
│         CommandRouter (命令路由)                │
├─────────────────────────────────────────────────┤
│      StateSynchronizer (状态同步)               │
├─────────────────────────────────────────────────┤
│        MemorySystem (记忆管理)                  │
├─────────────────────────────────────────────────┤
│    WorkflowIntegration (工作流整合)             │
├─────────────────────────────────────────────────┤
│         AgentSystem (统一接口)                  │
├─────────────────────────────────────────────────┤
│       WorkflowEngine (工作流编排)               │
├─────────────────────────────────────────────────┤
│       AgentExecutor (执行调度)                  │
├─────────────────────────────────────────────────┤
│       AgentRegistry (Agent 管理)                │
├─────────────────────────────────────────────────┤
│      Agent Definitions (32 个)                  │
└─────────────────────────────────────────────────┘
```

### Agent 分类

**6 个功能 Lane，32 个专业 Agent**:

1. **Architect Lane** (架构设计)
   - architect, designer, planner

2. **Executor Lane** (代码实现)
   - executor, refactorer, migrator

3. **Reviewer Lane** (代码审查)
   - api-reviewer, security-reviewer, quality-reviewer, style-reviewer, test-reviewer, performance-reviewer

4. **Optimizer Lane** (性能优化)
   - build-fixer, dependency-manager

5. **Documenter Lane** (文档编写)
   - docs-specialist, content-writer

6. **Tester Lane** (测试)
   - testing-specialist

7. **Specialist Lane** (专业领域)
   - backend-specialist, frontend-specialist, database-specialist, devops-specialist, mobile-specialist, ml-specialist, data-specialist, git-specialist

8. **Support Lane** (支持功能)
   - analyst, debugger, explore, orchestrator, product-manager, ux-researcher, team
