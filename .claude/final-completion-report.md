## 📚 使用示例

### 快速开始

#### 1. 执行单个 Agent
```javascript
import { createAgentSystem } from './src/agents/agent-system.js';

const system = createAgentSystem();

// 执行代码探索
const result = await system.execute('explore', {
  target: 'src/',
  depth: 'medium'
});
```

#### 2. 执行工作流
```javascript
// 代码分析工作流
await system.executeWorkflow({
  name: 'Code Analysis',
  tasks: [
    {
      id: 'explore',
      agentId: 'oh-my-claudecode:explore',
      input: { target: 'src/' }
    },
    {
      id: 'review',
      agentId: 'oh-my-claudecode:quality-reviewer',
      input: { code: '...' },
      dependencies: ['explore']
    }
  ]
});
```

#### 3. 使用命令
```javascript
import { createAgentCommand } from './src/commands/agent-command.js';

const agentCmd = createAgentCommand();

// 列出所有 Agent
const agents = agentCmd.list();

// 执行 Agent
const result = await agentCmd.run('explore', ['{"target":"src/"}']);
```
