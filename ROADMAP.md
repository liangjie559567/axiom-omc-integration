## 🚀 规划中

### v2.2 - 增强功能（预计 2-3 周）

#### 1. 更多内置命令
**优先级**: 高

- [ ] `config` 命令 - 配置管理
  - `config get <key>` - 获取配置
  - `config set <key> <value>` - 设置配置
  - `config list` - 列出所有配置
  - `config reset` - 重置配置

- [ ] `plugin` 命令 - 插件管理
  - `plugin list` - 列出已加载插件
  - `plugin load <path>` - 加载插件
  - `plugin unload <name>` - 卸载插件
  - `plugin reload <name>` - 重新加载插件

- [ ] `agent` 命令 - Agent 管理
  - `agent list` - 列出所有 Agent
  - `agent info <name>` - Agent 详情
  - `agent execute <name> <task>` - 执行 Agent

#### 2. 更多钩子集成
**优先级**: 中

- [ ] `CommandExecute` 钩子
  - 命令执行前后触发
  - 支持命令拦截和修改

- [ ] `AgentDispatch` 钩子
  - Agent 调度前后触发
  - 支持 Agent 选择和配置

- [ ] `FileChange` 钩子
  - 文件变化时触发
  - 支持自动格式化、lint 等

- [ ] `GitOperation` 钩子
  - Git 操作前后触发
  - 支持自动提交、推送等

#### 3. 命令系统增强
**优先级**: 中

- [ ] 交互式命令支持
  - 命令行交互
  - 用户输入提示
  - 选择菜单

- [ ] 命令历史
  - 记录执行历史
  - 支持历史查询
  - 支持重新执行

- [ ] 命令自动补全
  - Tab 补全
  - 参数提示
  - 智能建议

#### 4. 文档增强
**优先级**: 高

- [ ] CommandSystem 详细使用指南
- [ ] 命令开发教程
- [ ] 钩子开发教程
- [ ] 最佳实践文档
- [ ] 故障排除指南
